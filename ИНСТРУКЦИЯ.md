# 📖 Инструкция по запуску Telegram-бота для агентства недвижимости

---

## 📁 Файлы проекта

```
realestate_bot/
├── bot.py            — основной код бота
├── sheets.py         — работа с Google Sheets
├── config.py         — настройки (токен, районы)
├── requirements.txt  — зависимости
└── credentials.json  — ключ Google (создадите сами)
```

---

## ШАГ 1 — Создайте Telegram-бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Введите название бота (например: `Квартиры АН`)
4. Введите username (например: `moy_realty_bot`)
5. Скопируйте **токен** — он выглядит так:
   ```
   1234567890:ABCdefGHIjklMNOpqrSTUvwxyz
   ```
6. Вставьте токен в `config.py` в строку `BOT_TOKEN`

---

## ШАГ 2 — Создайте Google Таблицу

1. Откройте [Google Sheets](https://sheets.google.com)
2. Создайте новую таблицу (название: "База квартир")
3. Скопируйте **ID таблицы** из адресной строки:
   ```
   https://docs.google.com/spreadsheets/d/  <<< ВОТ ЭТО >>>  /edit
   ```
4. Вставьте ID в `config.py` в строку `SPREADSHEET_ID`

---

## ШАГ 3 — Настройте Google API

1. Откройте [Google Cloud Console](https://console.cloud.google.com)
2. Создайте новый проект (кнопка вверху → "Новый проект")
3. В меню слева: **API и сервисы → Библиотека**
4. Найдите и включите **Google Sheets API**
5. Найдите и включите **Google Drive API**
6. В меню слева: **API и сервисы → Учётные данные**
7. Нажмите **+ Создать учётные данные → Сервисный аккаунт**
8. Введите любое имя, нажмите "Создать и продолжить", затем "Готово"
9. Нажмите на созданный сервисный аккаунт
10. Вкладка **Ключи** → **Добавить ключ → JSON**
11. Скачается файл — **переименуйте его в `credentials.json`**
12. Положите `credentials.json` в папку с ботом

### Важно! Дайте доступ к таблице:
1. Откройте скачанный `credentials.json`
2. Найдите строку `"client_email"` — скопируйте email
3. Откройте вашу Google Таблицу
4. Нажмите **Настройки доступа** (кнопка "Поделиться")
5. Вставьте email сервисного аккаунта
6. Выберите роль **Редактор** → Готово

## ШАГ 4 — Установите Python и зависимости

### На Windows:
```bash
# Установите Python 3.10+ с сайта python.org
pip install -r requirements.txt
```

### На Mac/Linux:
```bash
pip3 install -r requirements.txt
```

---

## ШАГ 5 — Запустите бота

```bash
python bot.py
```

или на Mac/Linux:
```bash
python3 bot.py
```

Если всё правильно, увидите:
```
Бот запущен...
```

## 🎯 Как пользоваться ботом

### 🔍 Поиск квартиры:
- Нажмите **🔍 Поиск квартиры**
- Выберите район (или "Все районы")
- Укажите цену (мин/макс) или пропустите
- Выберите количество комнат
- Укажите нужна ли мебель
- Выберите тип ремонта
- Бот выдаст все подходящие квартиры

### ➕ Добавить квартиру:
- Нажмите **➕ Добавить квартиру**
- Заполните данные по шагам
- Подтвердите сохранение
- Квартира сразу появится в Google Таблице

### 📋 Все квартиры:
- Показывает все квартиры в базе (до 15 штук)

## ⚙️ Настройка районов

В файле `config.py` найдите список `DISTRICTS` и добавьте/уберите нужные районы:

```python
DISTRICTS = [
    "Юнусабад",
    "Мирзо-Улугбек",
    # добавьте свои районы...
]
```

## 🐛 Частые проблемы

**"ModuleNotFoundError"** → запустите `pip install -r requirements.txt`

**"Invalid token"** → проверьте токен в `config.py`

**"SpreadsheetNotFound"** → проверьте ID таблицы и доступ сервисного аккаунта

**Бот не отвечает** → убедитесь что файл запущен и не закрыт терминал
